# Descarga Automática INFORME MAEMP

Aplicación Streamlit que descarga automáticamente informes MAEMP desde Permoda Midasoft en modo headless (sin interfaz gráfica visible). Incluye programación periódica y notificaciones.

## Características

- ✅ Ejecución headless (Chrome sin interfaz)
- ✅ Descargas automáticas renombradas con fecha/hora
- ✅ Scheduler: ejecuta periódicamente (cada N minutos)
- ✅ Notificaciones en tiempo real
- ✅ Historial de ejecuciones
- ✅ Logs en vivo en Streamlit

## Requisitos

- Python 3.8+
- ChromeDriver compatible con tu versión de Chrome
